# Personal Profile Rules

Applies on personal machines (bars, trip laptops).

## Projects

- Personal projects live in `~/projects/`
- Media at `~/media/`, torrents at `~/media/torrents/`
- Key projects: topstopstops (ttt/ttf aliases), dadmsg

## Tools

- Ghostty as primary terminal
- Zed as primary editor, nvim as secondary
- mpv for media, yt-dlp for downloads
- ollama for local AI models

## Style

- Prefer `c` or `claude` alias for Claude Code CLI
- Use `nv`/`v` for neovim, `zed` for Zed
