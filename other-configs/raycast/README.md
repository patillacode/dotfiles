# Raycast settings

The `raycast.rayconfig` file can be imported into your raycast app via the
"Import Settings and Data" command within raycast.

Same for exporting your setting, run raycast and look for the "Export Settings & Data" command

Save your export in the `other-configs/raycast` folder as `raycast.rayconfig` to overwrite the current export and that is it!

You can now import it anywhere else.
